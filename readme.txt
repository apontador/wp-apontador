== Description ==

This plugin provides a widget to display your latest reviews on apontador on your website.

== Installation ==

1. Upload and extract the zip/tar.gz to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Locate the "Apontador Settings button" and click on it to configure the plugin
4. Click the "request authorization Link, you'll be redirected to apontador (you must have an apontador account, create one if you don't! [Apontador](http://www.apontador.com.br) )
5. Apontador will ask you whether you want to allow this application to be able to use your data. Please allow the application to act on your name. This is required in order to fetch information from your account.
